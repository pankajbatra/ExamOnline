Configure ExamOnline

Please follow the instructions carefully.

1.

Once you have correctly created a  new test the test will appear in the control panel and is ready for configuration.
  

[Start Test]
you can start a test once you have added the sufficient number of questions for the test in the database.If the number of questions in the database is less than the minimum number of questions than you will not be able to start a test.Once you successfully started a test the status of the test will become Active.An active status indicates that the test is ready to be conducted and the test takers will be able to appear for the corrosponding test.
 

 [Stop Test]
you can stop a currently active test by using this control.Once a test has been stopped the status of the test will be changed to Inactive.An inactive test cannot be conducted and will not be available to the test takers.However, the administrator can start a test by using the start control.
 

[Delete Test]
The administrator should exercise this control only when he wants to delete the corrosponding test completely.This control should be used only when the test is to be stopped permanently.For stopping a test temporarily the administrator should use the Stop control.Tests once deleted cannot be recovered again.
 

[Manage test]
The administrator can Add,Edit and Delete questions for all the subjects in the corrosponding test using this control.It will also show the number questions present in the database and also the minimum number of questions required for the particular subject.
 

2.

 Manage Test Controls are explained below.
 

[Add Questions] - The administrator can use this control to add a question to the desired subject.  

[Edit Questions] - The administrator can use this control to edit a question from the desired subject.

[View Questions] - The administrator can use this control to view all the questions present in the subject.

[Delete Questions] - The administrator can use this control to delete a question from the corrosponding subject.
